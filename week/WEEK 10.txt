--------------------------------------------------------
PHASE 2 â€” OPEN KUBERNETES DASHBOARD
--------------------------------------------------------

Step 5: In the same CMD window, type:
        minikube dashboard

        (If it shows a security popup, click **Allow / Continue**)

Expected: Kubernetes dashboard UI with cluster details.

--------------------------------------------------------
PHASE 3 â€” CREATE NGNIX DEPLOYMENT & SERVICE
--------------------------------------------------------

Step 7: Go back to Command Prompt window
        Type:
        kubectl create deployment mynginx --image=nginx

Step 8: Check deployment and pods
        kubectl get deployments
        kubectl get pods

Step 9: Expose deployment as service
        kubectl expose deployment mynginx --type=NodePort --port=80 --target-port=80

Step 10: Check service port
        kubectl get service mynginx

--------------------------------------------------------
PHASE 4 â€” SCALE DEPLOYMENT TO MULTIPLE PODS
--------------------------------------------------------

Step 11: In CMD, type:
        kubectl scale deployment mynginx --replicas=4

Step 12: Verify 4 pods created
        kubectl get pods

--------------------------------------------------------
PHASE 5 â€” ACCESS NGINX WEBPAGE
--------------------------------------------------------

Step 13: In CMD, type:
        kubectl port-forward svc/mynginx 8081:80

Step 14: Open browser manually (Chrome)
        In address bar type:
        http://localhost:8081

Expected Output: **Nginx Welcome Page**

(KEEP CMD OPEN â€“ do not close)

--------------------------------------------------------
PHASE 6 â€” OPTIONAL: USING MINIKUBE SERVICE URL
--------------------------------------------------------

Step 15: If port-forward fails, type:
         minikube tunnel

         minikube service mynginx --url

Step 17: Copy URL and paste in Chrome

--------------------------------------------------------
PHASE 7 â€” DELETE PODS & SERVICE (CLEANUP)
--------------------------------------------------------

Step 18: In CMD, type:
         kubectl delete deployment mynginx

         kubectl delete service mynginx

         minikube stop

PHASE 8 â€” INSTALL & RUN NAGIOS MONITORING USING DOCKER
--------------------------------------------------------

Step 21: Open **Docker Desktop** (confirm running)

Step 22: Open **Command Prompt** again

Step 23: Pull Nagios image

         docker run --name nagiosdemo -p 8888:80 jasonrivers/nagios:latest

Step 25: Open browser (Chrome)
         Type in address bar:
         http://localhost:8888

Step 26: Login:
         Username: nagiosadmin
         Password: nagios

Step 27: Click on:
         **Hosts**
         **Services**
         **Service Health**
         **Alerts**
         **Host Groups**

(Click each tab â†’ take screenshot)

PHASE 9 â€” STOP & REMOVE NAGIOS
--------------------------------------------------------

Step 28: Return to Command Prompt
         Stop Nagios container:
         docker stop nagiosdemo

Step 29: Remove container:
         docker rm nagiosdemo

Step 30: Remove image (optional):
         docker rmi jasonrivers/nagios:latest
