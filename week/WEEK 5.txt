PHASE 1 â€” BASIC DOCKER REDIS CONTAINER (OFFICIAL IMAGE)

Step 4: Run Redis container in background mode
    docker run --name my-redis -d redis

    docker ps

Step 6: Open Redis CLI inside the running container
    docker exec -it my-redis redis-cli

Step 7: Set a value in Redis
    SET name "Alice"

    GET name

    exit

Step 10: Stop container
    docker stop my-redis

Step 11: Restart same container
    docker start my-redis

Step 12: Verify running container
    docker ps

Step 13: Stop container before deletion
    docker stop my-redis

Step 14: Remove container
    docker rm my-redis

Step 15: Remove Redis image
    docker rmi redis
